## 오늘 학습한 내용 중 가장 기억에 남는 것들을 몇 가지 적어볼까요? 📝

---

👉 (이곳에 작성해주세요~)

```Scss
$color-palette: #dad5d2 #3a3532 #375945 #5b8767 #a6c198 #dbdfc8;

@each $color in $color-palette {
  .icon-#{$color} {
    background: $color;
    width: 20px;
    height: 20px;
    border-radius: 50%;
  }
}

```

- #color가 숫자로 시작되면 에러가 나므로 유의할 것!
