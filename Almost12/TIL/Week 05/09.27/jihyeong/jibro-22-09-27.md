## 오늘 학습한 내용 중 가장 기억에 남는 것들을 몇 가지 적어볼까요? 📝

---

👉 (이곳에 작성해주세요~)

```CSS
/* 이미지 vertical-align은 resets에 추가 */
img {
	vertical-align: top;
	/* 또는 vertical-align: bottom; */
}

/* x축으로 이미지 반복되게 할 것 */
.ground {
	background-repeat: repeat-x; 
}

```

이미지는 기본적으로 인라인 요소이므로 여기서 여우는 display: block을 이용하여 블록레벨 요소로 변환하였다.
